      <h1 class="article__title h-1">Politique de confidentialité.</h1>
      <p class="article__dek lede">
        RareRadar.fr ne collecte aucune donnée personnelle identifiante. Le seul outil de mesure
        utilisé est Cloudflare Web Analytics, conçu pour respecter la vie privée des visiteurs
        sans cookie persistant ni adresse IP identifiable.
      </p>
    </header>

    <div class="prose">

      <h2>1. Données collectées</h2>
      <p>
        Aucun formulaire d'inscription, aucun compte utilisateur, aucune saisie de données
        personnelles n'est requis sur ce site. Aucun cookie publicitaire (Google Ads, Facebook
        Pixel, etc.) n'est posé.
      </p>
      <p>
        Seul <strong>Cloudflare Web Analytics</strong> est utilisé pour mesurer l'audience. Ce
        service&nbsp;:
      </p>
      <ul>
        <li>n'utilise aucun cookie persistant ni traceur côté navigateur&nbsp;;</li>
        <li>n'identifie pas individuellement les visiteurs&nbsp;;</li>
        <li>n'enregistre pas les adresses IP de manière identifiable (anonymisation appliquée par Cloudflare)&nbsp;;</li>
        <li>se limite à des données agrégées&nbsp;: nombre de pages vues, sources de trafic, pays.</li>
      </ul>

      <h2>2. Durée de conservation</h2>
      <p>
        Les statistiques anonymes Cloudflare Web Analytics sont conservées <strong>6 mois</strong>
        conformément aux paramètres du service, puis automatiquement supprimées.
      </p>

      <h2>3. Vos droits (RGPD)</h2>
      <p>
        Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez
        des droits suivants&nbsp;:
      </p>
      <ul>
        <li><strong>Droit d'accès</strong>&nbsp;: obtenir confirmation qu'un traitement vous concerne.</li>
        <li><strong>Droit de rectification</strong>&nbsp;: demander la correction de données inexactes.</li>
        <li><strong>Droit à l'effacement</strong>&nbsp;: demander la suppression de vos données.</li>
        <li><strong>Droit d'opposition</strong>&nbsp;: vous opposer au traitement de vos données.</li>
      </ul>
      <p>
        Pour exercer ces droits&nbsp;: <a href="mailto:contact@rareradar.fr">contact@rareradar.fr</a>.
        Réponse sous 30&nbsp;jours conformément aux dispositions légales.
      </p>

      <h2>4. Sous-traitants</h2>
      <p>
        <strong>Cloudflare, Inc.</strong> est sous-traitant technique (analytics anonymes + CDN +
        hébergement DNS). <strong>GitHub, Inc.</strong> est sous-traitant technique pour
        l'hébergement statique. Aucune donnée n'est vendue ou partagée à des fins commerciales.
      </p>

      <h2>5. Liens d'affiliation</h2>
      <p>
        Le site contient des liens vers des programmes d'affiliation (Amazon Partenaires, eBay
        Partner Network). Ces liens transmettent un identifiant de partenaire pour le calcul des
        commissions, sans collecte de données personnelles supplémentaire de la part de
        RareRadar. Voir la page <a href="/disclosure/">Transparence</a> pour le détail.
      </p>

      <h2>6. Modifications</h2>
      <p>
        Cette politique peut être mise à jour à tout moment. La date de dernière modification
        est indiquée ci-dessous.
      </p>
